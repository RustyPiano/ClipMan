# ClipMan v1.8.1 更新日志
> Windows 平台专项优化版本

## 🐛 缺陷修复

### Windows 平台问题修复
- **CSP 错误**：修复开发者工具连接被阻止的问题，添加 WebSocket 支持
- **版本显示**：修正设置页面版本号显示错误（v1.7.0 → v1.8.1）
- **设置持久化**：修复设置保存后不生效的问题，确保所有配置项正确保存
- **存储位置显示**：实现获取当前数据存储路径的 API，正确显示存储位置
- **窗口最小化**：修复最小化窗口后无法通过快捷键或托盘菜单重新唤起的问题

## 🚀 改进与优化

### 跨平台体验
- **快捷键显示优化**：快捷键预设按钮根据操作系统动态显示
    - macOS: ⌘⇧V、⌘⇧C、⌥V
    - Windows: Ctrl+Shift+V、Ctrl+Shift+C、Alt+V
- **文件管理器文本**：按钮文本根据平台调整
    - macOS: "在Finder中打开"
    - Windows: "在资源管理器中打开"

### 代码质量
- 移除未使用的设置管理器方法，代码更简洁
- 统一窗口显示逻辑，添加 `unminimize()` 支持
